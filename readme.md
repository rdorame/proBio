# ProBio Web Page
Web page designed for ProBio on August-December Semester of 2016.


### Prerequisites

You must have installed the last version of [Laravel](https://laravel.com/docs/5.3)


### Installing

```
git clone https://github.com/rdorame/proBio.git

composer update

php artisan serve

```
